## 如何开始

### 单片机

使用单片机开发软件KEILμVision，打开Project_STC编译，使用串口工具下载进单片机，即可开始工作。

### PC端引导程序

打开Project_pc文件夹，运行python文件，请注意依赖可能需要自行下载。

请注意，运行PC端引导程序需要维持串口通信（即您的单片机需要连接在电脑上），而维持单片机运行则仅仅需要将程序下载至单片机并维持单片机通电即可。
